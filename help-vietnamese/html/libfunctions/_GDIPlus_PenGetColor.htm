_GDIPlus_PenGetColor

Gets the pen color

#include <GDIPlus.au3>
_GDIPlus_PenGetColor ( $hPen )

Parameters

$hPen Handle to a pen object

Return Value

Success: the pen color.
Failure: -1 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).
