_WinAPI_GUIDFromString

Converts a string GUID to binary form

#include <WinAPI.au3>
_WinAPI_GUIDFromString ( $sGUID )

Parameters

$sGUID GUID in string form

Return Value

Success: a $tagGUID structure.
Failure: sets the @error flag to non-zero.
